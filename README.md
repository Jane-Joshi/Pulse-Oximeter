# Pulse-Oximeter
A low-cost Arduino-based Pulse Oximeter developed to measure **Blood Oxygen Saturation (SpO₂)** and **Heart Rate (BPM)** using optical sensing. The system processes signals from IR and Red LEDs through a photodiode and displays the readings on an OLED screen in real time.

## Features

* Real-time SpO₂ Monitoring
* Heart Rate (BPM) Measurement
* OLED Display Output
* Portable & Low-Cost Design
* Battery Powered
* Easy to Build and Operate

## Hardware Used
* Arduino Uno
* OLED Display (SSD1306)
* IR LED
* Red LED
* Photodiode
* BC547 Transistors
* LM7805 Voltage Regulator
* 9V Battery
* Resistors

##  Software
* Arduino IDE
* Embedded C++

## Results
The developed prototype successfully measures:
* Blood Oxygen Saturation (SpO₂)
* Heart Rate (BPM)
The readings are displayed on the OLED screen in real time.
